# ドキドキドギド

## 製品コンセプト

本プロジェクトでは冒険の主体は人間プレイヤー。
AIドギドは状況を盛り上げ、マイクラ世界をより楽しむための視点の提供を行う。

## AIと発見しよう！マイクラ世界の新しい見方！

ちょっと怖がりなAI相棒「ドギド」は、
Minecraftの冒険中にふいに川柳を詠みます。

でも、ドギドの句はちょっぴり下手くそ。
プレイヤーは「どこが変？」「どう直す？」とツッコミながら、
言葉を観察し、表現を工夫していきます。

自分で詠んだ句も保存できます。
洞窟、朝日、クリーパー、ウーパールーパー。
Minecraftの世界が、そのまま表現の材料になります。

---

# 少し教育向けに寄せた版

## Minecraftで、言葉の力を育てよう。

怖がりなAI相棒「ドギド」は、プレイ中の発見や出来事をもとに川柳を詠みます。

ただし、ドギドの句はいつも完璧ではありません。
プレイヤーは「どこが変か」「どう直すとよいか」を考えながら、AIの句を添削できます。

この体験を通じて、観察力、語彙力、表現力、そしてAIの出力をうのみにしない批判的思考を育みます。

プレイヤー自身が詠んだ句も保存でき、Minecraftの冒険を言葉の記録として残せます。
