# 環境由来の敵仕様

この文書は、2026-05-31 時点で固まった `水辺 / 燃焼 / 粉雪` まわりの敵個別仕様をまとめたものです。

モンスター単体の種類だけではなく、`地形や状態のせいで危険度や演出が変わるケース` をここで扱います。

## 1. 共通原則

- 水辺の遠隔敵は、通常時より脅威度を高く見る
- 射線が通る浅瀬の遠隔敵は、`背後 クリーパー` を除き他の敵コメントより優先する
- 日中燃焼コメントは演出的においしいが、水辺の遠隔敵が見えている時はそちらを優先する
- 粉雪に完全に隠れている敵は、プレイヤーが確認できる状態になるまでサーチ対象から外す

## 2. 浅瀬の遠隔敵

### 対象

- `skeleton`
- `stray`
- `trident drowned`
- その他、`水で足を取られにくい or 水越しに射線を通せる` 遠隔敵

### 定義

- `in_water = true`
- ただし `完全水没` ではない
- かつ、プレイヤーに対して実質的に射線が通る

実装上は、`浅瀬 = プレイヤー側だけ不利になりやすい水際戦` とみなす。

### 優先度

- 優先度は `high`
- 射線が通るなら短い警告を優先する
- 他のモンスターへの通常コメントは後回し
- 例外は `真後ろから迫るクリーパー`

### プレイヤーが弓 / クロスボウを使える場合

- 条件:
  - `player.held_item` が `bow` または `crossbow`
  - または、即使用可能な遠隔武器を持っていると判断できる
- 期待する反応:
  - 「遠くから撃てるなら倒せるで！」
  - 「足が取られるから近づかんほうがええ！」
- 出力方針:
  - 高優先の短文
  - 実装上は pre-cache して即時再生してよい

### プレイヤーが弓 / クロスボウを使えない場合

- 基本方針:
  - `嫌な相手` として強めに扱う
  - 倒しに行くより、逃げや位置変えを勧める

#### 8 ブロック以内

- 反応:
  - 「あー！あかんあかん！ぶが悪いわ！」
- 方針:
  - 即時警告
  - 通常の単体敵コメントより優先

#### 8 ブロック超

- 日中なら:
  - 「あああ！[対象モンスター]が燃えよらん！！」
- 共通追加候補:
  - 「倒しにくいから、俺あいつのこと嫌いやねん！！」
  - 「浅瀬に[対象モンスター]！悪夢のような存在やわ！」
- 抑制:
  - 状態が変わらない限り `20 秒` ごとに警告

## 3. 完全に水の中の遠隔敵

### 定義

- `in_water = true`
- かつ `完全水没`
- かつ 遠隔攻撃タイプ

### 基本方針

- `脅威ゼロ` ではない
- ただし浅瀬や陸上より圧は落ちる
- `いることだけ覚えとこか` のトーンで留める

### 期待する反応

- 「あっ[モンスター名]が…まー、射程が落ちるし、いることだけ覚えとこか・・・」

### 抑制

- 同じ個体または同状態の同種へのコメントは `5 分` 抑制
- `浅瀬` や `陸上` に戻ったら抑制を打ち切る
- 状態変化時は、完全水没メッセージを途中で打ち切って新状態のメッセージへ移る

## 4. 燃える近接敵

### 対象

- `zombie`
- `zombie_villager`
- その他、`日中に燃える近接型`

### 基本方針

- 燃焼状態は `助かった` 演出として使う
- ただし遠隔水辺脅威が見えている時は、そちらを優先する
- 遠隔敵がいない場合だけ、プレイヤーに最も近い対象の燃焼状態を出力する
- それ以外の燃焼個体は抑制する

### 反応の流れ

- まず通常の存在コール:
  - 例: `[モンスター] n 体おるで`
- その直後に燃焼リアクション:
  - `[燃えとる]`
- 途中で水に入ったら:
  - 燃焼メッセージを止めて `[水に浸かった]`
- 再び燃え始めたら:
  - 水メッセージを止めて `[燃えとる]`

### 往復抑制

- `水に浸かる -> 燃える` を `2 回以上` 繰り返したら:
  - `[ ...忙しいやっちゃな ]` を出す
  - その後は `燃えた / 水に入った` の両方を `5 分` 抑制する

## 5. 粉雪の中の敵

### 基本方針

- 粉雪内のモンスターはサーチ対象外
- プレイヤーが現時点で確認できないなら、透視的には扱わない

### 再表示条件

- プレイヤーが粉雪を払った
- プレイヤー自身が粉雪の中に入った
- そのほか、プレイヤーが敵を視認できる状態になった

### 期待する反応

- 見えた瞬間に:
  - 「ぎゃー！」
- 近距離なら `ambush` と同格で扱ってよい

## 6. 必要観測項目

### すでに使えるもの

- `player.held_item`
- `inventory`
- `visual_threats[].in_water`
- `visual_threats[].on_fire`
- `visual_threats[].distance`
- `world.time_phase`

### 追加すると助かるもの

- `visual_threats[].submerged_state`
  - `none / shallow / submerged`
- `visual_threats[].ranged_line_of_sight`
- `visual_threats[].obscured_by_powder_snow`
- `derived.burn_state_flip_count`

## 7. 実装メモ

- 浅瀬遠隔敵の comment cooldown は `20 秒`
- 完全水没遠隔敵の state-stable suppression は `5 分`
- 燃焼 / 入水往復の `忙しいやっちゃな` 後も `5 分` 抑制
- 粉雪 reveal は `panic_cue` または `ambush cue` 扱いでよい
