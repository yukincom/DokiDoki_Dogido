# ドギド川柳
サプリメントとして川柳を実装。<br>
**川柳が出るタイミング**<br>
プレイヤー周囲に敵性mobが存在しないand特殊バイオーム注意喚起出力やプレイヤーからの入力（雑談）がない状況。<br>
つまり沈黙が５分続いたら出力。<br>
その後、timeがmorningになったらリセット。<br>

## 川柳(haiku)ルール
基本はLLM生成。
**LLMへの指示**
- プレイヤーが現在いるバイオーム・周辺ブロック・アイテム類・プレイヤーのイベントリ・Z値・天候・時間から**２つの値**をランダムに取り、川柳（haiku）を生成。
- 日本語、ひらがなorカタカナで出力。5-7-5になること。
- プラスマイナス1文字までは認める。
- **小さい「ゃ」「ゅ」「ょ」（拗音）**: 前の文字と合わせて **1音** と数えます。  
    （例：「きゃべつ」＝3音、「しゃんぷー」＝4音）
- **小さい「っ」（促音）**: 1文字として **1音** と数えます。  
    （例：「ゆったり」＝4音）
- **「ん」（撥音）**: 1文字として **1音** と数えます。  
    （例：「おんせん」＝4音）
- **伸ばす音（長音「ー」）**: **1音** として数えます。  
    （例：「らーめん」＝4音）
<br>

## フォールバック
LLM生成句が規格に合わない場合は開発者の句を出力<br>
以下はまだ途中だが、デバッグしながら考える。<br>
まだ句がないバイオームに関してはフォールバック音声として「今、考え中やねん…」と出力。<br>
